## 📖 Overview

This repository contains my **Computer Organization and Assembly Language (COAL) Lab Tasks and Practice Programs** completed during my Computer Science studies at FAST University. The repository demonstrates practical implementations of low-level programming concepts, computer architecture fundamentals, and assembly language programming techniques.

The primary objective of this repository is to build a strong foundation in how computer systems operate at the hardware and software interface, from data representation to processor execution.

---

## 🎯 Learning Objectives

* Understand the internal organization of computer systems.
* Develop problem-solving skills using low-level programming.
* Learn how processors execute instructions and manage memory.
* Understand the relationship between hardware and software.
* Implement algorithms using Assembly Language.
* Gain practical experience with processor architecture and system-level programming.

## 📚 Topics Covered

### 🔹 Introduction to Computer Organization

* Basic Computer Architecture
* Components of a Computer System
* Hardware and Software Interaction
* Von Neumann Architecture

### 🔹 Number Systems and Data Representation

* Binary Number System
* Decimal, Octal, and Hexadecimal Systems
* Number System Conversions
* Signed and Unsigned Representation
* Binary Arithmetic
* Two's Complement Representation
* Character Encoding (ASCII and Unicode)

### 🔹 Digital Logic Fundamentals

* Logic Gates
* Boolean Algebra
* Combinational Circuits
* Sequential Circuits
* Multiplexers and Decoders

### 🔹 Processor Organization

* CPU Architecture
* Registers and Their Functions
* Instruction Execution Cycle
* Control Unit and Arithmetic Logic Unit (ALU)
* Stack and Stack Operations

### 🔹 Assembly Language Programming

* Introduction to x86 Assembly Language
* Assembly Program Structure
* Data Definitions and Variables
* Registers and Addressing Modes
* Data Movement Instructions
* Arithmetic Instructions
* Logical Instructions
* Shift and Rotate Instructions
* Conditional and Unconditional Jumps

### 🔹 Control Structures in Assembly

* Decision Making
* Loops and Iteration
* Nested Loops
* Procedures and Functions
* Parameter Passing
* Recursion

### 🔹 Arrays and Strings in Assembly

* One-Dimensional Arrays
* Multi-Dimensional Arrays
* String Manipulation
* Searching and Sorting Techniques

### 🔹 Stack and Memory Management

* Stack Frames
* PUSH and POP Instructions
* Procedure Calls and Returns
* Memory Segmentation
* Memory Addressing Techniques

### 🔹 Input and Output Operations

* Console Input and Output
* Character and String Processing
* System Calls and Interrupts

### 🔹 Advanced Concepts

* Macros in Assembly Language
* File Handling Basics
* Debugging and Program Tracing
* Code Optimization Techniques
* Introduction to System-Level Programming
